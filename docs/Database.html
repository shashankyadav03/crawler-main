unfoldDB :: Connection -> IO () Source #

Method to retrive all the URLs from the database, parse and extract new links from these, and store those links on the database. This method could be made more efficient by remembering which URLs have already been processed, so that next time unfoldDB is called only newly retrieved urls should be processed.
